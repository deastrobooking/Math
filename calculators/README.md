## üöÄ Getting Started

### Python
```bash
cd python
python3 calculus_calculator.py
```

### Rust
```bash
cd rust
cargo run --bin calculus_calculator
```

### C++
```bash
cd cpp
g++ -std=c++17 calculus_calculator.cpp -o calculus_calc
./calculus_calc
```

### C
```bash
cd c
gcc -lm calculus_calculator.c -o calculus_calc
./calculus_calc
```
